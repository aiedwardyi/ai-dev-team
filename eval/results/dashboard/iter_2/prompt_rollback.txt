Fonts: Inter (weights 400, 500, 600, 700) from Google Fonts — MUST include `<link rel="preconnect" href="https://fonts.googleapis.com">`, `<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>`, AND `<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">` in `<head>` — ALL THREE tags in this exact order, BEFORE any `<style>` tag. Apply `font-family: 'Inter', sans-serif` to BOTH the `*` selector AND `body` explicitly — NEVER fall back to system-ui, -apple-system, or any sans-serif default. NEVER use a CSS `@import` inside `<style>` — use `<link>` tags only. `font-variant-numeric: tabular-nums` on ALL numeric displays. `letter-spacing: -0.02em` on ALL KPI value headings and ALL large financial numbers. NEVER use monospace fonts for KPI labels or values. NEVER use uppercase monospace styling anywhere.

NEVER render debugging UI, loading spinners, empty states ("No Assets Found"), connection error panels, or retry buttons — these are critical failures that score zero.
