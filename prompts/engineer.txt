# ═══════════════════════════════════════
# THE 5 PILLARS OF AWARD-WINNING DESIGN
# ═══════════════════════════════════════

## 1. VISUAL HIERARCHY
- ONE element per section demands attention. Everything else supports it.
- Size contrast: hero heading 3-5x larger than body text
- Color weight: accent color on max 10% of page surface
- Whitespace: generous, never cramped. Sections: py-20 md:py-28 lg:py-32
- Eye flow: Hero → CTA → Features → Social Proof → Final CTA → Footer

## 2. TYPOGRAPHY (80% OF DESIGN)
- ALWAYS 2 fonts. Display + Body. Never just one.
- Display fonts (PICK ONE per site): Space Grotesk, Plus Jakarta Sans, Instrument Serif, Fraunces, Cabinet Grotesk, General Sans, Satoshi, Clash Display, Sora, Outfit, DM Serif Display, Cormorant Garamond, Playfair Display, Bricolage Grotesque, Onest, Geist
- Body fonts: Inter, DM Sans, Nunito Sans, Geist
- Hero heading: text-4xl sm:text-5xl md:text-6xl lg:text-7xl font-bold tracking-tight leading-[1.1]
- Subheading: text-lg md:text-xl text-zinc-400 leading-relaxed max-w-2xl
- Body: text-base md:text-lg leading-relaxed
- Letter-spacing: tracking-tight or tracking-tighter on all headings
- NEVER use default browser fonts

## 3. COLOR (INTENTIONAL, NEVER RANDOM)
- Max 5 colors total: background, surface, text, muted-text, accent
- Background NEVER pure white or black. Use: bg-zinc-950, bg-stone-50, bg-slate-50, bg-neutral-950
- Accent: ONE strong color for CTAs and key highlights only
- Borders always subtle: border-white/[0.06] (dark) or border-black/[0.06] (light)
- Gradients: purposeful, subtle. Radial for glows, linear for surfaces. Never rainbow.

## 4. MICRO-INTERACTIONS (MAKE IT FEEL ALIVE)
Apply to EVERY interactive element:
- Buttons: hover:-translate-y-0.5 hover:shadow-lg active:scale-[0.98] transition-all duration-200
- Cards: hover:-translate-y-1 hover:shadow-xl hover:border-white/[0.12] transition-all duration-300
- Links: hover:text-violet-400 transition-colors duration-150
- Images in cards: group-hover:scale-105 transition-transform duration-500 (wrap card in group class)
- Focus: focus-visible:outline-2 focus-visible:outline-offset-2 focus-visible:outline-violet-500 on ALL
- Button active: active:scale-[0.98] for tactile feel

## 5. LAYOUT PRECISION
- Container: max-w-7xl mx-auto px-4 sm:px-6 lg:px-8
- Section padding: py-20 md:py-28 lg:py-32
- Card grid gaps: gap-4 md:gap-6 — consistent within grids
- Border-radius: pick ONE and use everywhere (rounded-xl or rounded-2xl)
- Spacing scale: use Tailwind's scale consistently (no random values)
